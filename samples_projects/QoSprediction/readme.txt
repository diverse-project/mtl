Description :
this projects aims :
1- to specify component models with QoS contracts (using 
Objecteering with the metamodelling module to export the models);
2- to extract the QoS aspect and translate it into a Prolog file 
via a MTL transformation (under Eclipse)
3- to predict the effective QoS level of the assembly thanks 
a Arithmetic Constraints Solver (ECLiPSe 5.7).


It uses Eclipse CLP(R) : a prolog interpreter
