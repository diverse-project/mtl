In order to compile this project in eclipse, you need to have a user lib named poseidon_libs.
In this directory you have different versions of this package of libraries depending on 
your configuration.
In order to use it : 
	simply go in the Preferences/Java/Build Path/User Libraries 
	and import it with the name "poseidon_libs".
