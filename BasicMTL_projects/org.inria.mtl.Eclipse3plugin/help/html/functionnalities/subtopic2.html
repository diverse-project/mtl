Create a New Eiffel File

To create a new Eiffel File:
-click on this icon in the EfE toolbar::
-or go in the /EfE/Create a New Eiffel File menu
-or file/new/create a new Eiffel File (The EfE perspective must be active)

Before this, if you want, you can select in the navigator the project who must contain the new file .

The wizard is displayed:



Choose in the container field the destination folder for your new file.
Enter the file name to create.
Then select what type of file you want to create:
-empty file
-hello world file
-prestructured file (file with skeleton)

Click Ok, the file is displayed in the editor.
