Documentation Generation

EfE use the Short SmartEiffel function to generate your source code documentation.
To do this, you must configure this command in the /windows/preferences/EfE Preferences Page menu.


You can choose the documentation format output (See the SmartEiffel Documentation for more details) and the short function parameters.

Once you have choose the settings, click on ok.
Choose the source file you want to generate the documentation in the navigator.
Then click on the following button:
The documentation is generated in the doc directory of the project.
