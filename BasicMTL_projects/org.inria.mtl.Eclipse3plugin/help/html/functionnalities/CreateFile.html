Create MTL File

To create an XML document, select File → New → MTL File ....

If you are working in a perspective other than MTL, you may not see MTL File... as a choice in the File → New menu. In that case, select Other.... This will show a dialog that allows you to select MTL and MTL File and click Next to proceed to the File Name and Location page. For convenience in the future, you can easily configure the perspective to show these menu items by default.
Figure 1: Menu New File

Whichever path you choose, it will take you to the following page:
Figure 2: FileName and Location

Select a location from the tree and enter the name of your new document. In the example, we have selected Test and entered the name MyFile. If you do not enter an extension, .mtl will be automatically appended. Therefore, the name of this document will be MyFile.mtl

Click Finish .
