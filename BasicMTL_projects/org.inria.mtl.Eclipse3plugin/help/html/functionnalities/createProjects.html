Create MTL Projects


Every MTL file you edit in Eclipse must be part of some project. So the first thing you must do, assuming you don't have existing projects already, is create a project.
select File → New
Alternatively, you can use the New Project  icon in the toolbar, as shown below.

Figure 1: MTL New project creation Menu



                                                                                                                                        

There are two basic ways to do this.

  • Create a new, empty project in a new directory in the workspace
  • Create a new project in an existing directory in the file system

Each task is discussed in a separate section.
