The Pretty Command

EfE use the Pretty SmartEiffel function to format the source code of your classes.
To do this, configure this command in the /window/preferences/EfE Preferences Page menu


You can choose the output format of pretty (see the SmartEiffel documentation for more details).

Then click ok.
Choose the file you want to format in the navigator.
Then click on this button:
Your code is formated!
