Create a new Eiffel Project

To create a new Eiffel Project
-click on this icon in the EfE toolbar::
-or go in the /EfE/Create a New Eiffel Project menu
-or file/new/create a new Eiffel Project (The EfE perspective must be active)

The wizard is displayed. Enter the project name and click on Ok.
The project is added in the navigator. It contains 4 folders:


bin

This folder contains the binaries files.

bytecode

This folder contains the bytecode java generated by SmartEiffel.

doc

This folder contains the documentation generated files.

src

This folder contains your source code.

Althrought it's not advising, you must not respect this structure if you don't want it.
