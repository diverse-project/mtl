Compilation

You can choose the compilation options un the /window/preferences/EfE Preference Page/Compiler menu.


First you can choose one of the three SmartEiffel compiler avaible and enter compilation parameters.
For more details, see the SmartEiffel help.

Once the compiler is configured, select a .e file to compile in the navigator and click on one of this two buttons: . The fist allow you to compile without the parameters you entered in the preferences. The second uses these parameters. If you want, you can use theses two functions throught the EfE menu.

At the end of the compilation, a report is displayed in the console. If there are no errors, the compiled files are added to the project, in the bin directory, if you chose compile or in the bytecode directory if you chose the compile_to_jvm compiler.
