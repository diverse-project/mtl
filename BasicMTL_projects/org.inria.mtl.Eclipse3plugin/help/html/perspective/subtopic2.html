Preferences

The EfE configuration is accessible in the "window/preference/Eiffel Preferences Page" menu:




Compiler Options



In this section you can manage all SmartEiffel Compiler Options.
Choose the SmartEiffel compiler to use among the three avaibables and enter the compilation parameter to use.
These parameters must have the following format: -param1 -param2


Documentation Generation Options



In this section you can manage the Documentation Generation options. You can select the documentation format to generate and Short command parameters.


Pretty Command Options



In this section you can manage the pretty command options, with you can format your source code.


Syntax Enlighting Colors



In this section you can choose the syntax enlighting colors.
