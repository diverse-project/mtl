Thank you for using MTL For Eclipse!

MTL for Eclipse is an Eclipse Plugin, with you can develope MTL applications under the Eclipse environment.

MTL For Eclipse provides a set of platform extension contributions (views, editors, perspectives, etc.) that collectively streamline the process of developing MTL models transformations inside the workbench.

It uses MTL Compiler (download here)

MTL Plugin offers a graphic interface for all the MTL Compiler functions:

  • Syntax coloring. Coloring is user-configurable through preferences. Coloring is performed for ordinary MTL File.
  • Code assist. Assistance is provided for elements like keywords and functions
  • Validation. A file may be validated on demand, when the document is saved or in the background while you edit. 
  • Outline view. The outline view shows the structure of the elements in your file. By default, the outline is dynamically synchronized with editing.
